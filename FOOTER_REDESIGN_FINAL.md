## ✨ MICROINTERACTIONS

### 1. **Link Hover - Dot Appearance**
```css
/* Default: hidden dot */
opacity-0

/* Hover: dot appears */
group-hover:opacity-100 transition-opacity
```

### 2. **Icon Box Hover**
```css
/* Default */
bg-moss-500/10

/* Hover */
group-hover:bg-moss-500/20 transition-colors
```

### 3. **Arrow Translation**
```css
/* Default */
<ArrowRight />

/* Hover */
group-hover:translate-x-1 transition-transform
```

### 4. **Social Icon Scale**
```css
/* Default */
text-lg

/* Hover */
group-hover:scale-110 transition-transform
```

### 5. **Badge Effects**
```css
/* Default */
opacity-60 grayscale-[30%]

/* Hover */
group-hover:opacity-100 group-hover:grayscale-0
```
